# Inventory3D

Прототип системы инвентаря в 3D, выполненный как тестовое задание в Unity.

## О проекте

Задача прототипа — показать, как может выглядеть инвентарь, когда предметы существуют не только в списке, но и в 3D-сцене.  
Это небольшая демка, где важны структура и идея, а не финальный арт.

В проекте:

- 3D-сцена с окружением и предметами
- интерфейс инвентаря поверх сцены
- привязка предметов в инвентаре к 3D-объектам
- аккуратное разнесение логики по префабам и скриптам

## Управление
ЛКМ двигаем объекты распределяя их по заранее определенным ячейкам.
После они видны в инвентаре при наведении.
Клик по объекту в инвентаре - выбрасывает объект
